# 天候パターンと出現条件

## 概要
FF14のフィールドはエオルゼア時間に基づく天候遷移を持つ。特定の魚種やFATEは天候が出現条件となっているため、天候パターンの法則性を理解することは実用上重要。

## 天候遷移の基本

- 天候は**エオルゼア時間8時間ごと**に変化（0:00, 8:00, 16:00）

## 天候パターンの観測例

| 地域 | 天候パターン | 出現確率 |

| *(ダミー: コルシア島)* | 快晴→快晴→雨 | *(要計測)* |
